## ❓ FAQs KHI BẮT ĐẦU

### Q: Mình không có dữ liệu chính xác (FTIA data, test results). Làm sao?
**A:** Không vấn đề! Bạn có thể:
```
1. Estimate dựa vào kinh nghiệm + customer feedback
2. Ghi rõ source: "Theo feedback khách hàng..." hoặc "Theo test nội bộ..."
3. Tính tí tí: Nếu bạn nói "khô nhanh", hãy test = 
   "Quick dry khô trong ~30 phút vs cotton ~90 phút"
4. Thêm caveat: "Tùy điều kiện thời tiết, nhưng nói chung quick dry nhanh hơn"

Chính xác 100% không cần thiết - Có dữ liệu hơn không có!
```

### Q: Content cứ bị dài. Làm sao?
**A:** 
```
1. Chia nhỏ sections - Mỗi H2 = 1 topic
2. Dùng bullets / lists - Dễ scan
3. Dùng bảng (tables) - Compact, dễ so sánh
4. Mỗi paragraph 3-4 câu - Không quá dài

Dài nhưng rõ = Tốt cho AI scanning
Ngắn nhưng mơ hồ = Xấu cho cả người + AI
```

### Q: Làm sao biết có thêm dữ liệu không?
**A:** Check:
```
1. Customer reviews → Lấy feedback
2. Competitor products → So sánh
3. GOOGLE "chất liệu cotton test results" → Tìm dữ liệu
4. Internal data (bảo hành, returns) → Độ bền
5. Ask team → Có ai test sản phẩm trước?
```

### Q: Mình không tự tin viết. Làm sao?
**A:**
```
1. Copy template từ PRODUCT-UPDATE-EXAMPLES.md
2. Fill-in blanks (tên sản phẩm, lợi ích, dữ liệu)
3. Edit lại cho tự nhiên
4. Ask AI (ChatGPT) để improve

Template = 80% công việc, bạn chỉ fill details!
```
